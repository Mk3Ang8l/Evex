# README

## About

Evex Research is a desktop research tool for organizing web research, indexing content, and keeping everything in one place. Built with Wails (Go + React).

## Development

Run `wails dev` in the project directory for live hot-reload development. A dev server also runs on http://localhost:34115 for browser access to Go methods.

## Building

Run `wails build` to produce a redistributable production package.
